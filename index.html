Features

研究室の運用に必要な機能を、ひととおり。

F-01

在室ボード（全体・部屋別）

研究室全体と部屋ごとの在室状況をボード表示。今そこに誰がいるかをリアルタイムに確認できます。

F-02

出退勤・ステータス管理

出勤・退勤の打刻とステータス変更をその場で実行。メンバーの状態がすぐに反映されます。

F-03

勤怠履歴と管理者修正

勤怠の履歴を蓄積し、管理者による打刻の修正にも対応。記録の正確さを保てます。

F-04

日誌の作成・編集・閲覧

日々の作業記録を日誌として作成・編集・参照。研究室の活動を文章で残せます。

F-05

管理ツール

メンバー・部屋・表示順を管理者がまとめて設定。研究室の構成に合わせて運用できます。

F-06

監査ログ

すべての変更操作を監査ログとして記録。いつ・何が変更されたかを後から追跡できます。

F-07

タッチモニタ・ボードモード

タッチスクリーン専用のボードモードとキャリブレーション機能を搭載。壁掛けモニタでそのまま使えます。

F-08

バックアップ・エクスポート

SQLite・JSON・Markdown 形式でデータをエクスポート。記録の持ち出しや保全が容易です。

F-09

LAN内クローズド運用

外部公開せず研究室内の端末からのみ利用する前提。セッションベース認証で守られた閉じた環境で動作します。

05

Architecture

Raspberry Pi 1台で動く、閉じた研究室LANのなかで。

Raspberry Pi 上でフロントエンド（Vite 開発サーバー）とバックエンド（Uvicorn で FastAPI）を同時に起動し、研究室LAN内の端末からアクセス。データは SQLite と JSON / Markdown ファイルストアに保存し、セッションベース認証で保護。外部公開はしないクローズド構成です。
